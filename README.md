# Caculator_With_Assembly8086

this is a caculator that can caculate any expressions have:
          1. not negetive operands
          2. with these operators:
              1. ADD ( + )
              2. Multiply ( * )
              3. Minus ( - )
              4. Divide ( / )
Also this caculator can caculate expressions have paranthes and print the right answer.
This caculator can determine these errors:
          1. Invalid Charecter
          2. Unbalenced Paranthes
          3. Divide By Zero
